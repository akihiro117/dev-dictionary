fishでpathを追加する方法(ユニバーサル変数を使う場合)

以下のコマンドを実行する。 
      
        set -U fish_user_paths <追加したいpath> $fish_user_paths

fishでpathを削除する方法(ユニバーサル変数を使う場合)

1. 以下のコマンドで削除したいpathの順番を調べる。 
      
        echo $fish_user_paths | tr " " "\n" | nl
結果の例: 
      
        1 /hoge1/bin
        2 /hoge2/bin
        2 /hoge3/bin

2. 該当の箇所を削除する。 
      
        set -e -U fish_user_paths[1]
