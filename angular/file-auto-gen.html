Angularでのファイル自動生成方法

  • プロジェクト
              
            ng new <プロジェクト名>
  • module
    以下のコマンドを実行すると、src/app/<module名>直下にmodule.tsが作成されます。 
              
            ng g module <module名>
    routing.tsを同時に作成するには、--routingオプションを付ける。
  • component
    以下のコマンドを実行すると、src/app/<component名>直下にcomponent.ts、html、css、spec.tsが作成されます。 componentファイル群が作成されるパスを変更するには、component名の前にパスを書きます。 
              
            ng g component <component名>
  • service
    以下のコマンドを実行すると、src/app/<フォルダ名1>/<フォルダ名2> 直下に<service名>.service.ts、html、css、spec.tsが作成されます。 
              
            ng g service <フォルダ名1>/<フォルダ名2>/<service名>
